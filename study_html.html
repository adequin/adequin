HTML Study

After learning some basics and using templates in the class HTMAA I've decided to spend some more time following tutorials and learning basics so that this portfolio website becomes more manageable.

As the first study on this documentation page I plan to include:
  • summary of basic tutorials I followed
  • learn to create a containers I dont have to edit each menu and break each line manually
  • outline a template for study webpages, projects, and origami design documentation.
Some formatting is already included in the elements.html file and the generic.html file. You can see the code by right clicking and selecting "inspect". 

	Tips from w3 school html exercises:
	use 'pre' to preserve text linebreaks and spaces (although it changes the font too)
	
 styles 

	"style=color:blue;" is an example of css and html attribute
	"style=font-family:courier;" changes the font to courier
	"style=text-align:center;" to change align to center
	"style=font-size:50px;" to change font to 50 pixels
	"background-color" changes the background color

	
 formatting

	use strong to bold words (same as b)
	use em  for emphasis (same as  i  for italics)
	use  mark  to highlight (doesnt work here for some reason)
	use sub for subscript and  sup for superscript
	use del for strikethrough

	
quoting 

	use q to add quotation marks
	
use blockquote to quote and specify source with "cite tag". This doesn't render anything but is good practice and maybe used by search engines.

	use "bdo dir="rtl" (bi-drectional override)to make text go  right to left
	use abbr title="abbreviation" to display abbreviation when hovering over (called tooltip).
	
	
 colors 

	Colors can be specified using hex "#FFFFFF" is white 
	
	or using RGB "rgb(0,0,255)" is blue
	or using HSLA (hue, saturation, lightness, alpha/transparency) "HSLA(0,100%,0.5,0.5)"

	
 CSS 

	Surround css with style tags. Put tag to modify in front of brakets {}.
	body {background-color:blue} -> sets background to blue
	body {font-family:courier} -> changes body font to courier
	body {color:red;} -> sets body text color to red
	p {border: 1px solid yellow;} -> adds a border to paragraphs that use p tags

	
 Links 

	style="text-decoration:none;"> removes underline from link
	target="_blank"> makes the link open in a new window
	title="thing"> adds a tooltip to the link
	href="#mytext"> creates a link to a bookmark with id="mytext"

	
 Images 

	style="float:right;"> makes image float right of the paragraph its contained in
	you can create image maps using the area tag
	you can make a backgound image with style="background-image: url('myimage.png')"
	favicon code: link rel="icon" type="image/x-icon" href="/images/favicon.ico">
